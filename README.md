# Parkinson AI Project

This is a Django-based machine learning project for Parkinson disease prediction.

## Technologies Used

- Python
- Django
- NumPy
- Pandas
- Scikit-learn
- Joblib
- OpenCV
- Librosa
- Matplotlib

## How to Run

1. Create virtual environment
2. Activate virtual environment
3. Install requirements
4. Run migrations
5. Start server
