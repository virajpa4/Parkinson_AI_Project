About the AI Models
Spiral Drawing Model
  • Analyzes hand-drawn spirals
  • Detects tremors and inconsistencies
  • Uses contour analysis and feature extraction
  • Random Forest Classifier algorithm
  • Expected accuracy: ~85-92%
Voice Analysis Model
  • Analyzes voice recordings
  • Detects speech pattern abnormalities
  • Extracts MFCC, pitch, and jitter features
  • Gradient Boosting Classifier algorithm
  • Expected accuracy: ~80-88%
Symptom Quiz Model
  • Evaluates symptom questionnaires
  • Calculates risk scores
  • Uses SVM with RBF kernel
  • Considers motor and non-motor symptoms
  • Expected accuracy: ~75-85%
